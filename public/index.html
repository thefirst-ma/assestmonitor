📊 投资标的监控平台

支持加密货币、股票、黄金、外汇等多种资产监控

📡 数据源说明

₿ 加密货币 Binance API - 实时价格，支持搜索
📈 股票 Yahoo Finance - 全球股票，支持搜索（如：AAPL, 600519.SS）
🥇 贵金属 Metals API - 黄金、白银、铂金、钯金
💱 外汇 Exchange Rate API - 主要货币对

⚙️ 基础配置

建议设置为 900 秒（15分钟）或更长
当价格变动超过此百分比时触发通知

📧 邮件通知

Gmail 用户需要在账户设置中生成应用专用密码

🔔 Webhook 通知

从钉钉/企业微信群机器人设置中获取

📱 Telegram 通知

从 @BotFather 获取机器人 Token
