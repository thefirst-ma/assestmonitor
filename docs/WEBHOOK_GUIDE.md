# 🔔 Webhook 通知配置指南

## 概述

Webhook 通知支持钉钉、企业微信等即时通讯工具，适合团队协作场景。

---

## 一、钉钉机器人配置

### 1. 创建钉钉群

1. 打开钉钉 APP
2. 创建一个新群组（或使用现有群组）
3. 群名称建议：`投资监控通知`

### 2. 添加自定义机器人

1. 进入群聊，点击右上角"..."
2. 选择"群设置" → "智能群助手"
3. 点击"添加机器人" → "自定义"
4. 填写机器人信息：
   - 机器人名称：`投资监控`
   - 添加到群组：选择刚才的群
5. **安全设置**（三选一）：
   - ✅ **自定义关键词**：输入 `价格` 或 `警报`（推荐）
   - 加签：需要额外配置
   - IP地址：限制访问来源

6. 点击"完成"
7. **复制 Webhook URL**（格式：`https://oapi.dingtalk.com/robot/send?access_token=xxx`）

### 3. 在平台中配置

访问 http://localhost:3001，切换到"系统配置"：

```
Webhook 类型：钉钉机器人
Webhook URL：粘贴刚才复制的 URL
```

### 4. 测试

1. 点击"启用 Webhook 通知"开关
2. 点击绿色"测试"按钮
3. 检查钉钉群是否收到测试消息

### 5. 通知效果

```
📈 加密货币 比特币 价格上涨警报

原价格: $50000.00
当前价格: $52500.00
涨跌幅: +5.00%
时间: 2026/3/13 15:30:00
```

## 二、企业微信机器人配置

### 1. 创建企业微信群

1. 打开企业微信 APP
2. 创建一个新群组
3. 群名称建议：`投资监控通知`

### 2. 添加群机器人

1. 进入群聊，点击右上角"..."
2. 选择"群机器人" → "添加群机器人"
3. 填写机器人信息：
   - 机器人名称：`投资监控`
4. 点击"添加"
5. **复制 Webhook URL**（格式：`https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=xxx`）

### 3. 在平台中配置

```
Webhook 类型：企业微信机器人
Webhook URL：粘贴刚才复制的 URL
```

### 4. 测试

点击"测试"按钮，检查企业微信群是否收到消息。

---

## 三、自定义 Webhook 配置

如果你使用其他支持 Webhook 的服务（如 Slack、Discord、飞书等），可以使用自定义 Webhook。

### 1. 获取 Webhook URL

从你的服务中获取 Webhook URL。

### 2. 在平台中配置

```
Webhook 类型：自定义 Webhook
Webhook URL：你的 Webhook URL
```

### 3. 数据格式

自定义 Webhook 会收到以下 JSON 数据：

```json
{
  "assetId": "crypto:BTCUSDT",
  "assetName": "比特币",
  "assetType": "crypto",
  "oldPrice": 50000,
  "newPrice": 52500,
  "changePercent": 5.0,
  "timestamp": 1710316200000,
  "message": "📈 加密货币 比特币 价格上涨警报\n\n原价格: $50000.00\n当前价格: $52500.00\n涨跌幅: +5.00%\n时间: 2026/3/13 15:30:00"
}
```

你可以根据这个数据格式编写自己的处理逻辑。

## 四、常见问题

### Q1: 钉钉机器人提示"不包含任何关键词"

**原因：**
安全设置中配置了自定义关键词，但消息中不包含该关键词。

**解决方法：**
1. 返回钉钉机器人设置
2. 修改自定义关键词为：`价格` 或 `警报` 或 `通知`
3. 或者删除关键词限制（不推荐）

### Q2: 测试时提示"发送失败"

**可能原因：**
- Webhook URL 错误
- 机器人已被删除
- 网络连接问题

**解决方法：**
1. 检查 Webhook URL 是否完整
2. 确认机器人仍然存在
3. 重新创建机器人并获取新的 URL

### Q3: 企业微信收不到消息

**检查项：**
1. 确认 Webhook URL 正确
2. 确认机器人未被移除
3. 检查企业微信是否有消息限制

### Q4: 如何在多个群中接收通知？

**方法：**
1. 在每个群中分别添加机器人
2. 获取每个机器人的 Webhook URL
3. 目前平台只支持配置一个 Webhook
4. 如需多个，可以使用自定义 Webhook 转发

## 五、高级配置

### 钉钉加签验证

如果你选择了"加签"安全设置：

1. 获取密钥（Secret）
2. 需要修改代码实现签名算法
3. 参考钉钉官方文档：[自定义机器人接入](https://open.dingtalk.com/document/robots/custom-robot-access)

### 企业微信提及功能

在消息中 @特定成员：

```json
{
  "msgtype": "text",
  "text": {
    "content": "价格警报",
    "mentioned_list": ["@all"]
  }
}
```

目前平台暂不支持，可以通过自定义 Webhook 实现。

## 六、配置示例

### 钉钉配置示例

```env
WEBHOOK_ENABLED=true
WEBHOOK_URL=https://oapi.dingtalk.com/robot/send?access_token=abc123def456
WEBHOOK_TYPE=dingtalk
```

### 企业微信配置示例

```env
WEBHOOK_ENABLED=true
WEBHOOK_URL=https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=abc123def456
WEBHOOK_TYPE=wecom
```

## 七、最佳实践

1. ✅ 为不同用途创建不同的群组
2. ✅ 使用有意义的机器人名称
3. ✅ 设置自定义关键词提高安全性
4. ✅ 定期检查机器人是否正常工作
5. ✅ 不要将 Webhook URL 公开分享
